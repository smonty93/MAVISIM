Welcome to the Documentation for MAVISIM

MAVISIM is a tool to simulate images generated by the MCAO Assisted Visible Imager and Spectrograph (MAVIS) instrument. MAVIS is an instrument being designed for ESO's VLT AOF (Adaptive Optics Facility, UT4 Yepun) with plans to go on-sky around 2027.

Version 1.0

MAVISIM 1.0 utilises a Fourier approximation of the MAVIS PSF to simulate a monochromatic image of point sources.

!!! Current Capabilities

  1. Able to simulate full stellar field (eg. Milky Way Globular Cluster) from an input physical catalogue
  2. Optimised for astrometric studies specifically
  3. Models three major sources of astrometric error introduced by the AO system
  4. Tip-tilt residuals originating from uncorrected LO aberrations (dependent on the NGS constellation brightness and geometry) field variable
  5. HO aberrations originating from the LGS constellation and characteristics field variable
  6. Static field distortion originating from the AO module optics

Current Limitations

Monochromatic images only
Currently only compatible with an approximate model of the MAVIS PSF using the Fourier method
Only compatible with input point source catalogues only
